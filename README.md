# FOTOGRAPH

Portafolio personal de fotografía minimalista. Un espacio dedicado a la obra visual, donde la experiencia de navegación es inmersiva y libre de distracciones.

## Tecnologías

- **Framework:** React / Next.js
- **Estilos:** Tailwind CSS
- **Animaciones:** Framer Motion
- **Optimización:** Imágenes en formato .webp

## Estructura del Proyecto

```
/public/images      # Archivos de imagen optimizados
/src/components     # Componentes de UI (ScrollToTop, PhotoCard)
/src/data           # Curaduría de la galería (gallery.ts)
```

Características
Diseño Inmersivo: Scroll personalizado sin barras visibles.

Optimización: Carga eficiente de imágenes con lazy loading.

Navegación Fluida: Transiciones suaves y botón de retorno al inicio.

Zero-Clutter: Enfoque absoluto en la calidad visual.

Despliegue
Este proyecto está configurado para un despliegue rápido en Vercel:

Subir el repositorio a GitHub.

Conectar el repositorio a Vercel.

El proyecto se compilará automáticamente con npm run build.
